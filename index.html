TheSims.css

A “Design System” inspired by the 2000 life simulation game, The Sims.

This is a mostly classless CSS library which relies on Semantic HTML to work; classes are used only for component variants, grouping elements and modals. The code is adapted from 98.css.

Components

Text Input

Textarea

Button

Buttons in The Sims come in all shapes and sizes, most of which are not reproduced in this library. For now, there are two types: rectangular (the default) and skewed.

Skewed

Add the class skew to give a button a skewed appearance.

Button group

Group multiple buttons together using the button-group class.

Radio

Checkbox

Slider

Sliders can rendered by specifying a range type on an input element.

Modal

Create a modal by adding the class modal to an element.
